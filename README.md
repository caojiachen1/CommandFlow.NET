# CommandFlow.NET

CommandFlow.NET is an automation workflow tool built using .NET and WPF. It provides a graphical interface for designing and executing automation tasks such as mouse movements, keyboard inputs, loops, and more.

> **Deprecated:** This project is no longer maintained. Development has been moved to a new Rust-based implementation.

For the latest work and future updates, please visit the successor repository:
